# 🚗 Car Rental Management System

A Python-based Car Rental Management System built using **Object-Oriented Programming (OOP)** and **SQL** to manage car bookings, customers, and rental records efficiently.

---

## 📌 Features

* 🚗 Add new cars to the system
* 👤 Add customer details
* 📊 View available cars
* 🔄 Rent a car
* ✅ Return a car
* 📋 View customer rental history

---

## 🛠️ Tech Stack

* **Programming Language:** Python
* **Concepts Used:** OOP (Classes & Objects)
* **Database:** SQL Server
* **Interface:** Command Line Interface (CLI)

---

---

## ⚙️ How to Run

1. Clone the repository

```bash
git clone https://github.com/Anis8989/car-rental-system.git
```

2. Navigate to the project folder

```bash
cd car-rental-system
```

3. Run the application

```bash
python app.py
```

---

## 🗄️ Database Setup

* Ensure **SQL Server** is installed
* Update your connection string in `app.py`:

```python
SERVER=FAIJAN\\MSSQL
```

* Create required tables for:

  * Cars
  * Customers
  * Rentals
